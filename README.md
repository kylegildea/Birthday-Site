# Birthday-Site
For this assignment you are required to create a web-page that when loaded, prompts the user to enter their birth day (day, month and year).  When the user enters the date, the information should be processed by embedded PHP code and the user's age is displayed. You are free to be as creative as you can to implement the special greeting. 
Your submission should include:

At least 1 form
PHP code that processes the form information
At least 1 function that process the form information
At least 1 function call to the created function/s
Artistically designed web-page using appropriate theme, fonts and colors.
